# TIPFS Bootstrap instructions
* Fork Repo
* Edit install.sh:
  * change default DNS endpoint (default: test.ipfs.telosfoundation.io)
* Create DNS Endpoint
  * $NETWORK.$DNS_ENDPOINT
  * boot -- bootstrap nodes
  * key -- swarm key
  * golangv -- Go language version to use
  * ipfsv -- IPFS version to use
